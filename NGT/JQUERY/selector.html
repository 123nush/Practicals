jquery Selector

Tag Name as Selector

Represents a tag name available in the DOM,.

For example $('p')selects all paragraphs in the document.

Id as Selector

  • Represents a tag available with the given ID in the DOM.
  • For example $('#some-id') selects the single element in the document that has an ID of some-id

Class as Selector

  1. Represents a tag available with the given class in the DOM.
  2. For example $('.some-class') selects all elements in the document that have a class of some-class.
